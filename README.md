# java-daw
A command-line java-based daw. 
### Getting Started
Clone the project and enter its directory in a terminal. Type
```
java -jar jdaw.jar
```
This will enter an interface for the program with
```
i <filename>
```
  to import a wav file
```
p
```
  to play the wav file and
```
q
```
  to quit.
